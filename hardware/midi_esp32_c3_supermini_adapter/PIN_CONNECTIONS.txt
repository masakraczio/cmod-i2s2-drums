MIDI DIN -> ESP32-C3 SuperMini -> Cmod A7 exact pin connections
=================================================================

This is the plain wiring list for the Rev A adapter.

Ground naming
-------------

LOGIC_GND means the shared digital ground of:
- ESP32-C3 SuperMini GND
- 6N138 pin 5
- Cmod A7 GND
- C1 pin 2
- R3 pin 2

MIDI_SHIELD means DIN connector pin 2 only.
Do NOT connect MIDI_SHIELD / DIN pin 2 to LOGIC_GND on this MIDI input.

3V3 means the ESP32-C3 SuperMini 3V3 pin.
Use the ESP32 3V3 pin for the 6N138 logic side, R2 pull-up, and C1.


Connector J1: DIN-5 MIDI IN
---------------------------

J1 pin 4  -> R1 pin 1
J1 pin 5  -> U2 6N138 pin 3, LED_K
J1 pin 2  -> MIDI_SHIELD / chassis only, no connection to LOGIC_GND
J1 pin 1  -> not connected
J1 pin 3  -> not connected


MIDI input LED path
-------------------

R1 value: 220 ohm

R1 pin 1 -> J1 DIN pin 4
R1 pin 2 -> U2 6N138 pin 2, LED_A

U2 6N138 pin 2, LED_A -> R1 pin 2
U2 6N138 pin 3, LED_K -> J1 DIN pin 5

D1 value: 1N4148 reverse protection diode, antiparallel across 6N138 input LED

D1 cathode / stripe side -> U2 6N138 pin 2, LED_A
D1 anode / non-stripe side -> U2 6N138 pin 3, LED_K


6N138 logic side
----------------

U2 pin 8, VCC -> ESP32-C3 SuperMini 3V3
U2 pin 5, GND -> LOGIC_GND
U2 pin 6, OUT -> ESP32-C3 SuperMini GPIO20, MIDI_RX
U2 pin 7, EN/base -> R3 pin 1
U2 pin 1 -> not connected
U2 pin 4 -> not connected

R2 value: 2.2k to 4.7k pull-up

R2 pin 1 -> ESP32-C3 SuperMini 3V3
R2 pin 2 -> U2 6N138 pin 6 OUT and ESP32-C3 GPIO20 MIDI_RX

R3 value: 10k

R3 pin 1 -> U2 6N138 pin 7 EN/base
R3 pin 2 -> LOGIC_GND

C1 value: 100 nF decoupling capacitor, place near U2 6N138

C1 pin 1 -> ESP32-C3 SuperMini 3V3
C1 pin 2 -> LOGIC_GND


ESP32-C3 SuperMini
------------------

ESP32-C3 SuperMini 3V3  -> U2 pin 8, R2 pin 1, C1 pin 1
ESP32-C3 SuperMini GND  -> LOGIC_GND
ESP32-C3 GPIO20         -> U2 pin 6 OUT and R2 pin 2, MIDI_RX, 31250 baud input
ESP32-C3 GPIO21         -> Cmod A7 uart_rx, 115200 baud output
ESP32-C3 5V             -> optional board power input only

Recommended first bring-up:
- Power ESP32-C3 SuperMini from USB.
- Connect only ESP32 GND to Cmod GND.
- Connect ESP32 GPIO21 to Cmod uart_rx.
- Do not feed Cmod 3V3 into ESP32 3V3 until you deliberately decide on one power scheme.


Connector J2: To Cmod A7 UART
-----------------------------

J2 pin 1, uart_rx    -> ESP32-C3 GPIO21
J2 pin 2, GND        -> LOGIC_GND / ESP32 GND / Cmod GND
J2 pin 3, 3V3_OPT    -> optional, normally not connected for first tests
J2 pin 4, uart_tx_op -> optional future Cmod TX to ESP32 RX, normally not connected


Minimal breadboard test wiring
------------------------------

MiniLab DIN OUT -> J1 DIN MIDI IN
ESP32 USB       -> PC USB power/programming
ESP32 GND       -> Cmod GND
ESP32 GPIO21    -> Cmod uart_rx

Then program Cmod, load sample bank, flash ESP32 firmware, and play MIDI notes.
